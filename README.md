# rock-paper-scissors-game

Player vs. Computer
Each side chooses either Rock, Paper, or Scissors

Simple Rules:
1) Rock beats Scissors
2) Paper beats Rock
3) Scissors beat Paper

Game will continue for 5 rounds.
Let's see who is the Ultimate Winner!
